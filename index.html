📖Bracket Guide
B1
Precon
No GC, no combos, no MLD. Unmodified or lightly upgraded precons.
B2
Casual
No GC. Tutors sparse. Average precon power level with upgrades.
B3
Optimized
Up to 3 Game Changers. No MLD. Late-game combos only (8+ mana).
B4
High Power
4+ Game Changers or MLD or fast combos. Pre-cEDH territory.
B5
cEDH
Optimized to win as fast as possible. Fast mana + infinite combos.
GAME CHANGERS: Legal in all brackets. Bracket 1–2 prohibit them. Bracket 3 allows up to 3. Bracket 4+ unlimited.

How Power Level is Calculated

This tool uses a power level algorithm based on published Commander methodology. Because we can't access live card price/popularity data offline, we use a curated local database of card ratings, bracket flags, and type data. Banned list and Game Changers are accurate as of February 9, 2026.

Bracket Requirements (WotC — Updated Feb 2026)
1
Exhibition
No extra turns, no MLD, no 2-card combos, no Game Changers. Tutors sparse.
2
Core
No chaining extra turns, no MLD, no 2-card combos, no Game Changers. Avg precon level.
3
Upgraded
No MLD. Late-game combos only (8+ mana to assemble). Up to 3 Game Changers.
4
Optimized
No restrictions beyond the banlist. High-powered, not tournament cEDH.
5
cEDH
No restrictions. Built to win in the competitive metagame.

📊 Power Score

Power score uses card Impact ratings (popularity + cost proxies) multiplied by an Efficiency factor from average CMC. Lower CMC = higher efficiency. The 1–10 scale uses a curve: casual decks ~4–6, high-powered ~7–8, cEDH 8.5–10. Unknown cards default to Impact 5, CMC 3.
